# Ralph Progress Log
# Pearl Homeowner App Rebuild
# Started: 2026-01-20

## Codebase Patterns
- React Native project is in `pearl-app/` subdirectory
- Use `@react-navigation/bottom-tabs` for bottom tab navigation
- Use `@react-navigation/native-stack` for stack navigation
- Screens are in `pearl-app/src/screens/`, navigation in `pearl-app/src/navigation/`
- Reusable components are in `pearl-app/src/components/` with barrel export via index.ts
- Run typecheck with `npx tsc --noEmit` from pearl-app directory
- Export components via index.ts barrel files for cleaner imports
- Theme system is in `pearl-app/src/theme/` - use `useTheme()` hook to access theme values
- Always wrap components needing theme access inside ThemeProvider (done in App.tsx)
- RootNavigator manages app-level navigation (splash screen, auth flow, main app)
- Use `colors.border.default` for border colors (not `colors.border` which is an object)
