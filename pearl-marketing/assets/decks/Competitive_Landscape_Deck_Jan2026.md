## Slide 13: Kukun Alert (Template: Page 17 - 2-column comparison)
**Title:** Kukun (PICO Score) — HIGHEST THREAT

**01 What They Do**
- Property condition scoring (500-850 scale)
- Claims traditional estimates off by $20K-$50K
- Asks homeowners about updates (roof, HVAC, kitchen)
- Calculates actual vs. neighborhood value
- **Free forever for consumers**

**02 Why It Matters**
- Closest thing to Pearl's value proposition
- Direct competition on condition → value connection
- Lender API for B2B revenue
- User count unknown — harder to track but potentially dangerous
- **Covers Operations pillar only** (no Safety, Comfort, Resilience, Energy)
