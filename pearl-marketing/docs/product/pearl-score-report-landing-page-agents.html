REPORT FEATURES
What's in the Report
What's inside a Pearl SCORE™ Report.

The nation's first home performance profile — shareable, comparable, and designed to help you answer the questions more buyers are asking today.

icon
Overall Performance Score
Every report includes a 1–1,000 Pearl SCORE™ for the property. Use it to help clients evaluate their shortlist — or to show what sets a listing apart.
icon
Five Performance Pillars
Pearl SCORE™ measures how a home performs across five critical pillars — Safety, Comfort, Operations, Resilience, and Energy. Each pillar is scored individually, so you can see exactly where a property excels and how it compares to others in the same market.
icon
Shareable Link
Every report comes with a shareable link — send it to clients comparing properties, embed it in MLS remarks, or include it in listing packets.
icon
Certifications & Achievements
If a property has earned certifications from programs like ENERGY STAR, LEED, or Pearl, the report flags them automatically — giving you instant visibility into verified improvements.
icon
Estimated Operating Costs
Each report includes estimated monthly operating costs and how they compare to similar properties — so you can have real conversations about total cost of ownership and show clients you're looking out for them beyond the closing table.
